# 公安经侦工作日志管理软件 配色模块落地细节（经侦场景专属）

说明：本配色细节严格贴合经侦案件登记场景，兼顾公安工作的严肃性、权威性与软件实操的易用性，适配Win7/10/11全版本、离线使用场景，与之前报告/表格模板完美适配，所有配色均采用WEB安全色，确保不同设备显示一致，同时符合公安数据合规与视觉规范。

# 一、核心配色原则（经侦场景专属）

- 权威性优先：主色选用低饱和藏蓝色系，凸显经侦工作的严肃性、公正性，区别于普通企业软件的活泼配色，契合政务类软件视觉规范。

- 易用性适配：配色对比度严格达标（文本与背景≥4\.5:1），适配民警长时间填写登记、查看报告/表格的场景，减少视觉疲劳。

- 合规性适配：避免高饱和、艳丽色彩，不使用非规范色值，所有配色符合公安内网软件视觉管理要求，不出现违规色彩搭配。

- 统一性原则：全软件配色统一（报告模板、表格模板、软件操作界面），不出现跨模块色彩偏差，降低民警视觉适应成本。

- 离线适配：所有配色均为WEB安全色，无需联网加载特殊色值，确保离线环境下色彩显示无偏差，适配公安内网纯离线使用需求。

# 二、具体配色落地细节（分模块明确，可直接复用）

## （二）报告模板配色落地细节（贴合经侦办案归档规范）

- 标题区域：报告主标题（XX市公安局经侦支队 案件阶段性工作报告）采用\#1565C0（主色）、18px、加粗，下方添加1px粗的主色下划线，与公安公文规范保持一致。

- 基础信息表格：表格边框采用\#E0E0E0（中性边框色），表头背景色采用\#F5F5F5（浅米色），表头文本采用\#333333（深灰）、加粗；表格内容文本采用\#666666（中灰），涉案金额、立案编号等核心数据采用\#1565C0（主色）、加粗，便于快速识别。

- 正文区域：一级标题（一、案件基础信息）采用\#1565C0（主色）、16px、加粗；二级标题（（一）核心工作内容汇总）采用\#333333（深灰）、14px、加粗；正文文本采用\#666666（中灰）、14px，行高1\.6倍，确保阅读舒适。

- 提示与补充区域：“手动补充区域”采用\#FFC107（提示色）边框，提示文本采用\#999999（浅灰）、12px，与正文形成区分，同时不干扰整体视觉。

- 签字与批示区域：签字栏边框采用\#1565C0（主色），日期栏与签字栏对齐，确保符合公安公文归档规范，打印后清晰可辨。

## （三）表格模板配色落地细节（适配台账归档、多案件统计）

- 表格标题：“民警每日工作记录统计表”采用\#1565C0（主色）、18px、加粗，统计说明文本采用\#999999（浅灰）、12px，位于表格底部，不占用核心视觉空间。

- 表头区域：表头背景色采用\#1565C0（主色），表头文本采用\#FFFFFF（纯白）、14px、加粗，确保表头清晰、突出，便于快速定位列内容。

- 表格内容：奇数行背景色采用\#FFFFFF（纯白），偶数行背景色采用\#F5F5F5（浅米色），形成奇偶区分，便于快速浏览多条工作记录；“完成状态”列按状态配色（已完成：\#4CAF50、部分完成：\#FFC107、未启动：\#D32F2F），直观区分工作进度。

- 核心数据高亮：关联案件、案件类型、工作成果等核心信息，采用\#333333（深灰）、加粗，与其他文本形成区分；涉案金额、笔录份数等量化数据，采用\#1565C0（主色）、加粗，便于快速提取关键信息。

- 打印适配：表格打印时，自动取消奇偶行背景色区分，所有文本统一为\#333333（深灰），边框保留\#E0E0E0（浅灰），确保打印后清晰、整洁，符合经侦台账归档要求。

## （四）软件操作界面配色（与报告/表格模板呼应）

- 导航栏：顶部导航栏背景色\#1565C0（主色），导航文本\#FFFFFF（纯白）、14px、加粗，选中状态添加白色下划线，便于民警快速切换模块（案件管理、工作记录、报告生成）。

- 表单区域：输入框背景\#FFFFFF（纯白），边框\#E0E0E0（浅灰），获取焦点时边框变为\#1565C0（主色）；字段标签采用\#333333（深灰）、14px，必填项标签后添加\#D32F2F（错误色）“\*”标记，清晰提示。

- 按钮区域：提交、保存按钮采用\#1565C0（主色），文本\#FFFFFF（纯白）、14px、加粗，点击时背景色加深为\#0D47A1；重置、取消按钮采用\#E0E0E0（浅灰），文本\#666666（中灰）、14px，与核心操作按钮形成区分，避免误操作。

- 暗黑模式适配：暗黑模式下，背景色\#121212（深灰），主色\#1976D2（稍亮藏蓝），文本色\#E0E0E0（浅灰），边框色\#333333（深灰），确保夜间使用时不刺眼，同时保持配色统一性。

# 三、落地禁忌与注意事项

- 禁止使用高饱和、艳丽色彩（如亮粉、亮绿、亮橙等），避免破坏经侦工作的严肃性，不符合公安软件视觉规范。

- 禁止配色杂乱，全软件色系严格控制在3种（主色\+辅助色\+中性色），报告/表格模板不新增额外配色，确保视觉统一。

- 禁止文本与背景对比度不达标，尤其是核心数据、操作提示，需确保不同视力民警都能清晰阅读，符合可访问性设计要求。

- 禁止色值不统一，同一类型元素（如按钮、表头）必须使用同一色值，避免出现不同页面、不同模板配色偏差。

- 离线适配注意：所有色值均采用WEB安全色，不使用特殊色值或渐变效果，确保在Win7/10/11不同版本、不同设备上显示一致，无需联网加载。

- 归档适配注意：报告/表格模板配色需适配打印场景，避免浅色文本、浅色边框，确保打印后清晰可辨，符合经侦台账归档要求。

# 四、配色与报告/表格模板的呼应说明

1\.  报告/表格模板的配色与软件操作界面完全统一，主色、辅助色、中性色复用，确保民警在填写登记、生成报告、查看表格时，视觉体验一致，降低适应成本；

2\.  核心数据（立案编号、涉案金额、工作成果等）在软件界面、报告、表格中均采用主色加粗显示，形成视觉呼应，便于快速识别关键信息；

3\.  状态标记（已完成、未启动、重点案件等）配色统一，软件界面中的状态提示与报告、表格中的状态标记保持一致，确保信息传递连贯；

4\.  打印场景下，报告/表格自动适配打印配色，取消不必要的背景色，保留核心文本与边框，既符合经侦归档规范，又节省油墨。
